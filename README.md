# API Eind Project

Install laravel

`composer install`

`cp .env.example .env`

`php artisan key:generate`

`php artisan jwt:secret`

Vul je gegevens in voor de database in je .env

`php artisan migrate --seed`

Voor de API documentatie klink [hier](https://school.wedevelop4you.nl/doc)
